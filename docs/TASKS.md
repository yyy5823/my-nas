|--------|----------|
| **M1: 可运行** | 项目可在所有平台编译运行 |

| **M3: 可播放** | 视频/音乐基础播放功能完成 |
| **M4: 功能完整** | 所有核心功能开发完成 |
| **M5: 可发布** | 通过测试，准备发布 |
| **M6: 正式发布** | 上架各应用商店 |
